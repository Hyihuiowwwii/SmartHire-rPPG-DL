# SmartHire-rPPG-DL

A student-level deep learning based rPPG project for heart rate estimation using webcam/video input.

## Features
- Login/UI pages
- Face detection using deep learning
- ROI-based signal extraction
- Basic heart rate estimation
- Dataset/video processing backup

## Tech Stack
- Python
- Flask
- OpenCV
- MediaPipe
- NumPy
- SciPy
- Matplotlib
- TensorFlow

## Run
```bash
pip install -r requirements.txt
python app.py
